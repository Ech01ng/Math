## Almost all things to do with Matrices that I could Find

<H1>Cofactor of a Matrices</H1>

<H2>Formula:</H2>
C<sub>i j</sub> = (-1)<sup>i + j</sup> x | D<sub>i j</sub> |

<H3>Where:</H3>
C = Cofactor
i = Row
j = Column
D = Determinant
sub = subscript notation

<H2>Example:</H2>
Given ->

A=
|1 -1 -1|
|2 3 8 |
|-3 2 1 |

Find C<sub>1 1</sub>

<H2>Process:</H2>

D<sub>1 1</sub>=
|3 8 |
|2 1 |

C<sub>1 1</sub> = (-1)<sup>1 + 1</sup> x | D<sub>i j</sub> |

C<sub>1 1</sub> = (1) (3x1 - 8x2)

C<sub>1 1</sub> = 3 - 16

C<sub>1 1</sub> = -13
