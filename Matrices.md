Cofactor of a Matrices

Formula:
C<sub>i j</sub> = (-1)<sup>i + j</sup> x | D<sub>i j</sub> |

Where:
C = Cofactor
i = Row
j = Column
D = Determinant
sub = subscript notation

Example:
Given ->

A=
|1 -1 -1|
|2 3 8 |
|-3 2 1 |

Find C<sub>1 1</sub>

Process:

D<sub>1 1</sub>=
|3 8 |
|2 1 |

C<sub>1 1</sub> = (-1)<sup>1 + 1</sup> x | D<sub>i j</sub> |

C<sub>1 1</sub> = (1) (3x1 - 8x2)

C<sub>1 1</sub> = 3 - 16

C<sub>1 1</sub> = -13
